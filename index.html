Blockchain Inspector

Thanks to our own ten-year-old Artificial Intelligence engine, we allow you to profile bitcoin users. The key advantage of our solution is that it can easily translates human reasoning into rules and in so doing can transpose the experience of experts into a computer system capable of processing very large amounts of data.

How it works

Our Artificial Intelligence is already used by the French railroad company (SNCF) to simulate crowds or by the French car maker Peugeot for its autonomous car

Extract Bitcoin data

Our first (open source) tool blockchain2graph extracts transactions/addresses from a bitcoin core server and stores them in a neo4j database.

Create rules

Each rule can be created individually and enables the step-by-step creation of a complex analysis system. The system uses fuzzy logic to handle the concept of partial truth, where the truth value may range between completely true and completely false.

View data

Our user interface (work in progress) allows you to search for peoples, companies, bitcoin addresses and transactions too see what data you have and what the Artificial Intelligence has added
